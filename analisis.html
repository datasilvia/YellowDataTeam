Planteamiento del ejercicio

Se nos pide hacer un análisis del dataset de AirBnb.
Nuestro objetivo es averiguar cómo influyen las diferentes características en el valor del inmueble
para poder hacer un algoritmo que prediga su precio en función de ciertas característica

1. Definición del Data Set

Definición del Data Set

El data set escogido es scrapeado de la web de AirBnB (https://insideairbnb.com/get-the-data/) y fue descargado de la web OpenDataSoft. Este conjunto de datos tiene aplicado un filtro y muestra la información principal sobre las habitaciones disponibles en AirBnB en Madrid para un período de tiempo determinado.
Las columnas que posee son:

  • Location
  • Coordinates
  • Room ID
  • Name
  • Host ID
  • Room type
  • Room Price
  • Number of reviews
  • Minimum nights
  • Updated Date
  • Availibility
  • Rooms rent by the host
  • Number of reviews per month
  • Date last review

  3. Pasar Datos Categóricos a Numéricos en 'room type'.
    Los modelos de aprendizaje automático y estadísticos generalmente requieren datos numéricos como entrada, ya que las operaciones matemáticas subyacentes no pueden manejar directamente las etiquetas categóricas.
    Por lo tanto, es crucial convertir las variables categóricas en un formato numérico antes de entrenar el modelo. Para ello empleamos la técnica de Codificación de Etiquetas (Label Encoding). Asigna un número entero único a cada categoría. Por ejemplo, para la variable 'room type', las categorías 'Entire home/apt', 'Private room' han sido codificadas como ‘Private room': 0, 'Entire home/apt': 1.

6. Conclusiones

Suposiciones Validadas:

Relación precio-ubicación:

Ya sabíamos que el precio estaría estrechamente relacionado con la ubicación del inmueble. Esta suposición se validó, ya que los datos mostraron una clara correlación entre el precio y las zonas donde se encuentran los inmuebles. Las áreas más céntricas y turísticas mostraron precios más altos.

Precio por tipo de habitación:

Asumimos que el precio variaría según el tipo de habitación ofrecida (Room Type), y se observó que el precio medio es mayor para apartamentos enteros (entire home/apt) en comparación con habitaciones privadas (Private Room). Esta suposición se validó claramente con los datos analizados.

Suposiciones No Validadas:

Hipótesis sobre las características influyentes:

Supusimos que algunas características del inmueble y comodidades como puede ser de habitaciones serían factores determinantes en el precio. Tras nuestro análisis hemos visto que no tiene una relación directa en el precio.

Efecto de las reseñas:

Supusimos que las reseñas y calificaciones tendrían un impacto significativo en el precio, pero los datos mostraron que, aunque influyen, no tienen tanto peso como la ubicación del alojamiento.
